# skill-creator

引導使用者從零設計並撰寫一個符合 AgentKit 規範的 Skill，包含草稿撰寫、測試案例、評估迭代、描述優化等完整流程。

## 安裝

```bash
/plugin install skill-creator@agentkit-skills
```

## 使用方式

安裝後，在對話中說出類似以下的句子即可觸發：

- 「我想建立一個 skill 來幫助 code review」
- 「幫我設計一個 skill」
- 「把這個工作流程變成 skill」

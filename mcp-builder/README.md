# mcp-builder

引導開發高品質 MCP (Model Context Protocol) Server，涵蓋工具設計、TypeScript/Python 實作、測試評估四個完整階段。

## 安裝

```bash
/plugin install mcp-builder@agentkit-skills
```

## 使用方式

- 「幫我建立一個串接 GitHub API 的 MCP Server」
- 「我想做一個 PostgreSQL MCP Server」
- 「如何把外部 REST API 包裝成 MCP Server？」
